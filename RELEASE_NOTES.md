# CoverTool v2.0.0 版本说明

## 🎉 新功能

### 🖼️ 批量制作封面
- 上传图片并自定义文字、字体、颜色
- 支持加载系统字体和自定义字体上传
- 实时拖拽调整文字位置
- 阴影效果和文本组合

### 🎥 批量嵌入封面
- 将封面嵌入视频第一帧
- 支持多种分辨率抖转（1080×1080、1920×1080、1080×1920）
- 保留原始音频信息
- 高效批量处理

### ✂️ 删除前x帧
- 批量删除视频开头的指定帧数
- 灵活设置帧数
- 自动输出到output文件夹

## 🔧 技术改进

- ✅ 完全重构UI界面，采用Tailwind CSS
- ✅ 修复系统字体加载问题
- ✅ 优化视频处理性能
- ✅ 添加交互式教程系统
- ✅ 跨平台支持（macOS、Windows、Linux）

## 📦 系统要求

- **macOS 10.13+** (Intel & Apple Silicon)
- **Windows 10+**
- **Linux** (各主流发行版)
- **4GB RAM**
- **500MB 磁盘空间**

## 📥 安装方式

### macOS
1. 下载对应版本的 `.dmg` 文件
   - Intel版本：`多功能封面工具-2.0.0-mac.zip`
   - Apple Silicon版本：`多功能封面工具-2.0.0-arm64-mac.zip`
2. 双击DMG文件打开安装程序
3. 拖拽应用到 Applications 文件夹

### 从源代码构建
```bash
git clone https://github.com/Kirota233/CoverTool.git
cd CoverTool
npm install
npm start
```

## 🐛 已知问题

- 首次加载系统字体可能需要几秒
- 处理大型视频文件时需要足够的磁盘空间
