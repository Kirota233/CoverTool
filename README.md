# 多功能视频处理工具 - 使用指南

## 功能概览

你的应用现在集成了三个强大功能：

### 1️⃣ 批量制作封面
- **用途**：生成高质量的视频封面
- **步骤**：
  1. 选择基础底图（PNG/JPG）
  2. 自定义文字内容、字号、大小、颜色等
  3. 支持拖拽调整文字位置和大小
  4. **新功能** ✨ 选择输出文件夹（不必保存在桌面）
  5. 一键生成所有集数的封面

### 2️⃣ 批量嵌入封面
- **用途**：将封面图片嵌入到视频第一帧
- **步骤**：
  1. 选择原视频所在文件夹（必须包含 EP1.mp4, EP2.mp4 等）
  2. **新功能** ✨ 选择封面文件夹（包含 cover_EP1.png 等）
  3. 设置项目名和输出分辨率
  4. 一键批量合成
  5. 输出结果会保存到视频文件夹的 `output` 文件夹

### 3️⃣ 批量删除前x帧
- **用途**：快速删除视频开头的不需要的内容
- **步骤**：
  1. 选择原视频所在文件夹
  2. 设置要删除的帧数
  3. 一键批量删除
  4. 输出结果会保存到文件夹的 `output_removed_covers` 文件夹

## 新增功能详情

### ✨ 输出文件夹选择
- 在"批量制作封面"页面，现在可以选择输出文件夹
- 不需要固定保存到桌面，可以选择任何位置
- 默认仍然是桌面的"生成封面图_Output"文件夹

### ✨ 封面文件夹选择
- 在"批量嵌入封面"页面，现在需要指定封面文件夹
- 支持选择任何包含 cover_EP*.png 的文件夹
- 提高了灵活性，支持多个封面库

### ✨ 启动界面
- 应用启动时显示三个功能卡片
- 每个功能都有详细说明
- 点击返回按钮可随时返回菜单

## 技术改进

- ✅ 多页面系统（launcher.html, index.html, remove-frames.html）
- ✅ 统一的 IPC 事件处理系统
- ✅ 支持自定义输出路径
- ✅ 更好的用户体验（返回菜单）
- ✅ 完整的错误处理和进度反馈

## 文件结构

```
CoverTool/
├── launcher.html          # 启动菜单
├── index.html             # 制作封面 + 嵌入封面功能
├── remove-frames.html     # 删除前x帧功能
├── main.js                # 主进程（新增多功能支持）
├── renderer.js            # 渲染进程（新增文件夹选择）
├── renderer-remove.js     # 删除功能渲染进程（新增）
├── package.json           # 项目配置
└── build/                 # 图标文件夹
```

## 启动应用

```bash
npm start
```

首次启动会显示启动菜单，选择你想要的功能即可。

---

**提示**：所有功能都支持返回菜单，点击左上角的"← 返回菜单"按钮即可。
